Forest Camping Site

This scene was made using a mixture of both 3D Studio Max and Unreal Engine 4, this was a simple scene design of a camping site in the forest and it was also my first time using Unreal Engine 4. This was not only to celebrate the awesome summer weather that the UK has been experiencing but also to allow me to try the engine out for an upcoming project that is currently getting worked on in the background.
