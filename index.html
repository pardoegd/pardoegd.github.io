This is a Prototype Home Page for Pardoe Games Designs!!

This is a site that will be used to shown my Level/Scene Designs that I will be making using the Unity Engine and possibly at a later date the Unreal Engine and even Cryengine. This is something that will be updated along side my YouTube, Flickr, Instagram and even Facebook Page. So please stand by as there will be a lot of awesome and exciting content coming soon to all of these platforms. Latest YouTube Video!!

Below is the latest Level Design Showcase Video, this was a combination of all of the seperate Levels/Scenes that I made using the Unity Engine in the later half of 2017. These were created using a mixture of 3D Studio Max for the model creation and Unity 2017 Personal for the overall game creation.
