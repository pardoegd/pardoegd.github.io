Level Designs

Welcome to my Level Designs Page, below you will be able to see all of my Level and Scene Designs that I have done using various Game Engines. Some of the 3D Models that you see were create by me using a mixture of both 3D Studio Max and Blender. The other assets that were used in these Scene/Level Designs were from the Unity Asset Store. The textures that I have used to build up the materials of the models were from a mixture Textures.com, arroway-textures and Dev-Assets. So, yeah, feel free to gander through the screenshots below and follow me on social media as that is the best way to keep up to date with all of my latest content. There are also 4K Downloads available for each of the screenshots shown below, these can be downloaded through my Flickr page, which is linked in the footer of each page in this site.



Unity Level Designs

The below section is filled with all of my Level Designs that I have done using the Unity Engine, I have used multiple versions of the Unity Engine below, from Unity 5 to Unity 2018.



Abandoned TARDIS

This was my first proper attempt at level design, the idea came from the reveal of the new (13th) doctor, the TARDIS Model was made by me using 3D Studio Max, however the Ivy was made using a script that is available for 3DS Max. The two TARDIS Textures were from Tumblr and Pinterest, other assets were from the Unity Asset Store.

Abandoned Stargate

This Scene Design was made using Unity and 3D Studio Max, this was my second level design. I decided to do this Scene/Level to celebrate the 20th Anniversary of Stargate SG1, as you can see I put a lot more detail into this scene and even created a bigger environment where there is more potential for more screenshots and possibly even some form of user interaction. The Stargate, Steps and DHD models was created by me again using 3D Studio Max, I also created some custom foliage for this level, using a mixture of both 3D Studio Max and Blender. Textures were from Tumblr and Pinterest, other assets were from the Unity Asset Store.

Garden Design

This was my third level design, and it was my first time using and experimenting with Lights in the Unity Engine, originally this was going to be a lot smaller of a scene than it is now, the original idea was that it was just going to be some ivy that had some LED Garden Lights mixed in with it, but then as you can tell it grew into the creation below. The Garden Design overall was a mixture of separate designs that I liked in certain gardens, that I thought would look good together. This was also created using the Vegetation assets that I made for the Abandoned Stargate Scene, however the flowers were created by me specifically for this level, again made using 3D Studio Max, other assets were from the Unity Asset Store.

Foggy Morning

The unique model that I made specifically for this scene is the Stone Bridge, however, I did use some of the previously created models to build up the scene.
